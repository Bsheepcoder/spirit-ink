**灵墨** 是一个 AI 驱动的粒子艺术引擎。输入任何文字，3000 个粒子将在 AI 的创意驱动下变换成各种有灵魂的形态。

## ✨ 功能特性

- **AI 形态生成** — 输入文字，AI 根据情感和意境生成粒子坐标和颜色
- **控制点插值** — AI 输出稀疏控制点，前端自动插值到数千个粒子
- **13 种预设形态** — 心形、人脸、五角星、海浪、爆发、漩涡、球体等
- **丝滑过渡动画** — 缓动粒子变形 + 鼠标排斥交互
- **实时配置** — 粒子数、控制点数、模型，全部在页面上可调
- **等待动画** — AI 思考时粒子自动变换为莫比乌斯环旋转
- **调试面板** — 完整查看 AI 响应、token 用量、耗时

## 🚀 快速开始

1. 用现代浏览器（推荐 Chrome/Edge）打开 `index.html`
2. 点击右上角 ⚙ → 粘贴智谱 API Key → 选择模型 → 保存
3. 在输入框输入任何内容 → 回车

## ⚙️ 配置说明

| 配置项 | 说明 | 范围 |
|--------|------|------|
| API Key | 智谱 AI API 密钥 | — |
| 模型 | 使用的 GLM 模型 | GLM-5 / 5.1 / 5-Turbo / 4.7 / 4.6 / 4.5 / 4.5-Air |
| AI 控制点 | AI 生成的点数（越多越精细，越慢） | 20–3000 |
| 渲染粒子数 | 最终显示的粒子总数 | 100–20,000 |

## 🛠 技术栈

- **Three.js** + WebGL 粒子渲染
- **EffectComposer**（Bloom 后处理）
- **智谱 GLM API** AI 生成
- 纯前端，单 HTML 文件，无需服务器
