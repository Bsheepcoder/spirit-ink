**Spirit Ink** is an AI-powered particle art engine. Type anything, and watch 3000 particles transform into expressive shapes driven by AI creativity.

## ✨ Features

- **AI-Driven Morphing** — Send any text, AI generates particle positions and colors to visually express emotions
- **Control Point Interpolation** — AI outputs sparse control points, frontend interpolates to thousands of particles
- **13 Preset Shapes** — Heart, Face, Star, Wave, Burst, Spiral, Sphere, and more
- **Smooth Transitions** — Easing-based particle animation with mouse repulsion
- **Real-time Config** — Adjust particle count, control point count, model, all from the UI
- **Wait Animation** — Möbius strip rotation while waiting for AI response
- **Debug Panel** — Full transparency into AI responses, tokens, timing

## 🚀 Quick Start

1. Open `index.html` in a modern browser (Chrome/Edge recommended)
2. Click ⚙ (settings) → Paste your Zhipu API Key → Select model → Save
3. Type anything in the input box → Hit Enter

## ⚙️ Configuration

| Setting | Description | Range |
|---------|-------------|-------|
| API Key | Zhipu AI API Key | — |
| Model | GLM model to use | GLM-5 / 5.1 / 5-Turbo / 4.7 / 4.6 / 4.5 / 4.5-Air |
| AI Control Points | Points generated by AI (more = detailed, slower) | 20–3000 |
| Render Particles | Final particle count displayed | 100–20,000 |

## 🛠 Tech Stack

- **Three.js** + WebGL for particle rendering
- **EffectComposer** (Bloom post-processing)
- **Zhipu GLM API** for AI generation
- Pure frontend, single HTML file, no server required
